## Steps
1) Exports

2) Token files

3) Templates
   - Slides, docs, code snippets, issue templates
4) Usage guidelines
   - Do/don'ts, safe areas, common mistakes
5) Output
   - Save to: `project-logs/product/brand-guidelines-{timestamp}.md`
